# ğŸ“› What is this plugin?

An experimental ESLint plugin using svelte-eslint-parser (opens new window).

# â“ Why?

Svelte (opens new window) has the official ESLint plugin the eslint-plugin-svelte3 (opens new window). The eslint-plugin-svelte3 (opens new window) works well enough to check scripts. However, it does not handle the AST of the template, which makes it very difficult for third parties to create their own the ESLint (opens new window) rules for the Svelte (opens new window).

The svelte-eslint-parser (opens new window) aims to make it easy to create your own rules for the Svelte (opens new window) by allowing the template AST to be used in the rules.

# â— Attention

The svelte-eslint-parser (opens new window) and the @ota-meshi/eslint-plugin-svelte can not be used with the eslint-plugin-svelte3 (opens new window).
